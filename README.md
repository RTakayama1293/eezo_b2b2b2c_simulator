# EEZO×BtoB 同時並行モデル コストシミュレーション

## 概要

北海道食材専門商社・新日本海商事が検討する「飲食店体験→EC購入」の信頼転移型マーケティング戦略について、国内BtoB営業とEEZO（EC）の同時並行モデルのコスト妥当性をシミュレーションするプロジェクトです。

## 分析目的

1. **3シナリオの経済性比較**
   - A: BtoB営業単独モデル
   - B: EEZO（EC）単独モデル
   - C: BtoB×EEZO同時並行モデル

2. **投資判断指標の算出**
   - LTV:CAC比率（目標4:1以上）
   - 損益分岐点（BEP）
   - 3年間累計ROI

3. **感度分析による不確実性の可視化**

## 主要パラメータ（初期設定）

### BtoB営業

| 項目 | 値 | 備考 |
|------|-----|------|
| 営業担当者数 | 1名 | 杉田さん想定 |
| 月間訪問件数 | 20件 | 週5件ペース |
| 成約率 | 10% | 2件/月 |
| 平均年間取引額 | 120万円 | 月10万円×12ヶ月 |

### EEZO（店舗経由EC）

| 項目 | 値 | 出典 |
|------|-----|------|
| 連携店舗数（初年度目標） | 10店舗 | 戦略レポート |
| 月間QRスキャン数 | 1,000回 | 戦略レポート |
| CVR（スキャン→購入） | 8% | ワイナリー実績ベース |
| 顧客単価 | 8,000円 | 戦略レポート |
| リピート率（3ヶ月以内） | 25% | 戦略レポート |

## 前提条件・制約

- 分析期間: 3年間（36ヶ月）
- 金額単位: 円（表示時は万円変換）
- インフレ率: 考慮しない
- 為替変動: 考慮しない（国内市場のみ）
